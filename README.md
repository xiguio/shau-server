# 鲨鱼CMS服务端

基于nodejs + eggjs + sequalize + mysql + redis

### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ npm run tsc
$ npm i --production
$ npm start
$ npm stop
```

### 客户端

客户端请移步 [shau-app](https://github.com/xiguio/shau-app)


[egg]: https://eggjs.org
